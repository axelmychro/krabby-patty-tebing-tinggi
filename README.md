# Kraby Paty Tebing Tinggi

### About the Business

A small local burger shop operating at night in Tebing Tinggi. This website
helps customers quickly view featured menu items and order via GoFood or
GrabFood.

### Project Goals

- Provide a fast, mobile-friendly landing page
- Highlight featured menu items with clear pricing
- Redirect users directly to GoFood or GrabFood for ordering

### Features

- GoFood and GrabFood CTA
- Simple, distraction-free layout
- Optimized for speed and mobile-first

### Tech Stack

- Vite + Preact
- Tailwind
- Vercel ([visit website](https://krabypaty-tebingtinggi.vercel.app))
